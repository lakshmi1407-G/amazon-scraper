# 🛒 Amazon Scraper Project

## 📌 Description

This project is a web scraping application that extracts product details from Amazon.
It collects important data like product name, price, and stores it for further use.

---

## ⚙️ Technologies Used

* Python 🐍
* BeautifulSoup / Requests
* HTML & CSS

---

## 🚀 Features

* 🔍 Extract product name
* 💰 Extract product price
* 📊 Store scraped data
* ⚡ Fast and simple scraping

---

## 📂 Project Structure

* `app.py` → Main application
* `main.py` → Execution file
* `templates/` → HTML files
* `DatabaseMongoDB.py` → Database handling

---

## ▶️ How to Run

1. Install required libraries
2. Run the main file:

   ```
   python main.py
   ```
3. View output in terminal / browser
